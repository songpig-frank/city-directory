# 📈 Progress & Roadmap — Tampakan Directory

This document tracks the evolution of the Tampakan Directory project.

## ✅ Completed Milestones

### **Phase 1: Foundation (Current)**
- [x] **Dark Mode "Clean Sweep"**: Complete overhaul of the CSS variable system. Replaced hardcoded Light Mode colors with a high-performance, semantic token system.
- [x] **Authentication Engine & Registration**: Resolved browser-locking issues for standard users. Standardized session keys and fixed database column mismatches.
- [x] **Global AEO Optimization**: Implemented site-wide and listing-level JSON-LD (Schema.org) for superior visibility in AI engine answer boxes.
- [x] **Repository Packaging**: Professional initialization on GitHub with `main` and `dev` branches.

## 🚧 In Progress

### **Phase 2: Management & Scale**
- [/] **Business Creator Crawler**: Finalizing the automated scraping and import mechanism for scaling listings.
- [ ] **Advanced Media Management**: Dynamic image resizing and WebP conversion on upload.
- [ ] **Interactive Maps Phase 2**: Clustering markers and "Near Me" geolocation support.

## 🗺️ Roadmap

### **Phase 3: Community & Engagement**
- [ ] **User Dashboards**: Allowing business owners to manage their own listings, photos, and promotions.
- [ ] **Review Moderation Suite**: A comprehensive dashboard for admins to approve/reject community feedback.
- [ ] **Automated Backups**: Nightly SQLite-to-Cloud sync.

## 🛠️ Tech Stack Evolution
- **v1.0**: Vanilla Stack (PHP/SQLite) — *Current Status*
- **v1.1**: Performance Tuning & PWA offline enhancements.
- **v2.0**: Transition to Headless API (optional future).
